- 레이아웃은 헤더, 좌/우측 패널, 본문, 푸터로 구성됨
- 헤더 및 헤더 아래 네비게이션은 유지
- 좌측 패널에는 3레벨 카테고리를 표시

- 카테고리 예시: IT(3) > backend(10) > Spring Boot(2)

- 채팅 input 아래로는 최신순 블로그 글 나열(커서 기반 무한 페이징)

- 채팅은 1회성 세션이며, 메모리 저장 없음, 새로고침 시 사라짐
